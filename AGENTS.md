# AGENTS.md

## 目的
このリポジトリにおける AI エージェント（汎用）の作業方針を定義します。

## 基本方針
- **会話言語**: 日本語
- **コメント**: 日本語
- **エラーメッセージ**: 英語
- **コミットメッセージ**: [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/) (description は日本語)

## 判断記録のルール
- 重要な技術的判断を行った場合は、その理由と代替案について記録を残すこと。

## 開発手順（概要）
1. プロジェクトのファイル構造 (`index.html`, `servers.json`) を理解する。
2. 必要に応じて `python -m http.server` 等でローカル確認環境を用意する。
3. 変更を実装する。
4. ブラウザ等で動作確認を行う。

## セキュリティ / 機密情報
- API キーやパスワードなどの機密情報をコードや `servers.json` に含めないこと。
- ログ出力に機密情報を含めないこと。

## リポジトリ固有
- 静的サイトであるため、ビルドプロセスはない。
- Vue.js と Vuetify は CDN から読み込んでいる。
